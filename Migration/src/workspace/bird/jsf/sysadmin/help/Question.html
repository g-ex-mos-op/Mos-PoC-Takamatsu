よくあるご質問
 
店舗種別とは何ですか？
 
店舗種別 説明
前年対象店 年度を通して前年対比が可能な店舗
　→当年度の前年対象店は、オープン日が前年度の４月１日以前である店舗
予算対象店 年度を通して予算対比が可能な店舗
　→当年度の予算対象店は、オープン日が
　　前年度の４月２日以降、当年度の４月１日以前である店舗
　※ただし、条件指定で「予算対象店」選択時は「前年対象店」も含まれる
新店 年度を通して予算対比ができない店舗≒今年度オープン店
　→当年度の新店は、当年度の４月２日以降にオープンする店舗
 
前年データ種別とは何ですか？
  ◆店舗種別：全店選択時
 
前年データ種別 説明
前年同月 前年同月内の同曜日売上と比較（月跨ぎが発生しない）
前提条件：当年／前年のどちらかがOPENしていれば、表示対象となる
  ◆店舗種別：前年対象店選択時
 
前年データ種別 説明
前年同月 前年同月内の同曜日売上と比較（月跨ぎが発生しない）
前提条件：当年OPENしていることが条件となり、前年実績があっても当年
               CLOSEしていれば表示対象外となる
前年同月営業日補正 前年同月内の同曜日売上と比較（月跨ぎが発生しない）
前提条件：当年OPENしていることが条件となり、当年／前年のどちらも
               売上が発生している場合のみ表示対象となる
前年同曜日 前年の同曜日を比較（月跨ぎが発生する）
前提条件：当年OPENしていることが条件となり、前年実績があっても当年
               CLOSEしていれば表示対象外となる
前年同日 前年の同日と比較
前提条件：当年OPENしていることが条件となり、前年実績があっても当年
               CLOSEしていれば表示対象外となる
